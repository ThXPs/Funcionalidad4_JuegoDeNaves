# Colisiones con nave y colisiones con proyectiles hacia enemigos
Este archivo se encarga de que el juego de naves ya establecido tenga una nueva funcion, esta trata de que si disparas hacia un
un enemigo, este desaparecerà. Y si los mismos enemigos llegan a chocar con la nave, estos le restaran uno de vida.

## Cómo probarlo
1. Compilar el archivo `funcionalidad4_colisiones.cpp`.
2. Ejecutar el archivo compilado.
3. Usa `A` y `D` para mover la nave.
4. Presione `Espacio` para disparar.
5. Verifique que cuando los proyectiles impactan a los enemigos, ambos desaparecen.
6. Presione `x` para salir.
